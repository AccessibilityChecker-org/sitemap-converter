Point it at any sitemap page — a WordPress index, a docs hub, a table of contents. It fetches the page, resolves every anchor, filters to same-domain links only, and returns a clean plain-text manifest. No guesswork, no crawling beyond page one — just the map.

STAGE 01
Fetch
One HTTP request, with a browser-like User-Agent.
STAGE 02
Parse
BeautifulSoup walks every anchor tag in the DOM.
STAGE 03
Filter
Drops mailto, tel, javascript, fragments, external hosts.
STAGE 04
Export
Sorted, deduped .txt — one URL per line.
